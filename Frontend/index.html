Step-by-step guide to using KORToEN
  • Click the 'Browse' button to select the image containing the text you wish to extract.
  • Once the image is selected, click the 'Process' button to initiate the processing.
  • The website will employ OCR technology to extract the text from the uploaded image.
  • The extracted text will then be translated into English using the Google Translate API.
  • You can view, copy, or download the translated text once the process is complete.
